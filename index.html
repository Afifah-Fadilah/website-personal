Materi Pembelajaran

Pertemuan 1: HTML

HTML digunakan untuk membuat struktur dasar halaman web.

    <!DOCTYPE html>
    <html>
    <head>
        <title>Halaman HTML Dasar</title>
    </head>
    <body>
        <h1>Halo, dunia!</h1>
        <p>Ini adalah paragraf pertama saya.</p>
    </body>
    </html>

Pertemuan 2: CSS

CSS digunakan untuk mengatur tampilan halaman web.

    body {
        font-family: Arial, sans-serif;
        background-color: #f0f0f0;
        color: #333;
    }

    h1 {
        color: blue;
    }

Pertemuan 3: JavaScript

JavaScript digunakan untuk membuat halaman web lebih interaktif. 

    document.addEventListener("DOMContentLoaded", function() {
        alert("Selamat datang di website saya!");
    });

Pertemuan 4: Pengenalan PHP

PHP adalah bahasa pemrograman yang digunakan untuk pengembangan web di sisi server. 

    <?php
    echo "Halo, dunia!";
    ?>

Pertemuan 5: Mengontrol Alur Program

Di PHP, kita bisa menggunakan percabangan untuk mengontrol alur program. 

    <?php
    $nilai = 85;
    if ($nilai >= 75) {
        echo "Anda lulus!";
    } else {
        echo "Anda harus mengulang.";
    }
    ?>

Pertemuan 6: Perulangan & Array

PHP mendukung berbagai jenis perulangan dan array.

    <?php
    $buah = ["Apel", "Jeruk", "Mangga"];
    foreach ($buah as $b) {
        echo $b . "<br>";
    }
    ?>
